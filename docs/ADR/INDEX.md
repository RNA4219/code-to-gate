## カテゴリ分類

- **基盤方針**: アーキテクチャ全体の前提・制約
- **検出ルール**: 品質検出ルールの設計判断
- **プラグイン**: プラグインSDK・サンドボックス関連
- **運用ガバナンス**: セキュリティ・SLO・運用統制

## 新規ADR作成手順

1. `docs/ADR/ADR-XXX-<title>.md` をテンプレートから作成
2. 本INDEX.mdに一覧を追加
3. README.mdのテーブルにも追加
4. PRテンプレートにADR番号を記載
