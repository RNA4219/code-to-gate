# Assurance Finding Vocabulary

このPhaseでは、将来のAssurance Smell Detector向けにrule IDとtag語彙だけを追加する。既存の`Finding.tags`で表現するため、artifact schemaは変更しない。

これらはbug断定ではなく、assurance gap、evidence gap、intent recovery gap、risk/test linkage gapなどのreview-required candidateを表す。検出ロジックとrelease判定は将来作業であり、code-to-gateは最終判断を行わない。
